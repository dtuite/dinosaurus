# Dinosaurus

Ruby wrapper for the [Big Huge Labs Thesaurus](http://words.bighugelabs.com/) API.

## Installation

Add this line to your application's Gemfile:

    gem 'dinosaurus'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dinosaurus

## Usage

require "dnosaurus"

results = Dinosaurus.lookup('word')
